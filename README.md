# H3-Cisco-Labs (Tribute til Jan)

Dette repository samler de labs, vi har lavet på vores Hovedforløb 3 (Netværk III & Netværksteknologi I).
Her finder du konfigurationer, øvelser og lab-scenarier fra forløbet —
blandt andet OSPFv3, eBGP/MP-BGP, MST, VRF-Lite og eksamenslignende opgaver.

Strukturen er inddelt i mapper per lab; hver mappe indeholder en kort
README og de nødvendige konfigurationsfiler, så man kan paste dem direkte på udstyret

Billedet nederst er af vores gode ven, der hedder Jan, vi har smidt det med bare
for sjov, fordi Jan altid ser sådan ud efter en lang dag med cisco konfigurationer.
